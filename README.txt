The zip file contains a subset of the data and models used in our project. The full dataset, including both data and models, is approximately 2GB, so only a sample is included here.

Sample Data Folder: This folder contains a sample of our actual data, consisting of 300 audio files used for training and approximately 30 files for testing.

test.ipynb: This Jupyter notebook contains the results of our testing.

test.py: This Python script contains the code used for testing the model.

train.py: This Python script outlines the process we used to train the model.

Speech_Downloads.ipynb: This notebook contains the code used for web scraping to collect the speech data.
